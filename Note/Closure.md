# 基础概念
rust中所谓的闭包可以与C++中的lambada表达式(匿名函数)进行类比。首先，我们需要复习一下C++中的lambada表达式用法

| 写法                   | 术语       | 含义                | Rust 对应概念        |
| :------------------- | :------- | :---------------- | :--------------- |
| `[]`                 | 空捕获      | 只能用参数，不能看外面       | 普通函数             |
| `[x]`                | 值捕获      | 拷贝一份 x (只读)       | `Copy` trait     |
| `[x] ... mutable`    | 值捕获 (可变) | 拷贝一份 x (可写)       | `let mut x = x;` |
| `[&x]`               | 引用捕获     | 操作原来的 x           | `&mut x` 或 `&x`  |
| `[=]`                | 隐式值捕获    | 只要用到的都拷贝          | (Rust 自动推断)      |
| `[&]`                | 隐式引用捕获   | 只要用到的都引用          | (Rust 自动推断)      |
| `[y = std::move(x)]` | 初始化捕获    | **移动** x 进 Lambda | move             |
